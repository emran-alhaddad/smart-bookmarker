🗂️ Filing Organize Strategy

Default is “Keep originals”. You can change this anytime.

Duplicate Removal: When using "Keep originals", duplicates within Smart Bookmarks are automatically removed after organization:
  • • Identifies identical URLs within Smart Bookmarks folder
  • • Keeps the oldest bookmark (by creation date)
  • • Removes newer duplicates
  • • Never affects original bookmarks outside Smart Bookmarks
  • • Normalizes URLs (removes tracking parameters, www, etc.)
